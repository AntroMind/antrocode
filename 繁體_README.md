## 🔥 為什麼選擇 AntroCode？ (Core Features)

### 🐣 我是一名 12 歲的開發者。這是我在考試周期間，利用 4 天時間挑戰極限完成的作品。
我把所有複雜的基礎設施全部拋棄，只留下最純粹的生產力與絕對的資料安全：

### 1. 🚀 極致輕量，開箱即用 (Zero-Install)
厭倦了每次都要 `npm install` 幾百 MB 的依賴？AntroCode 透過單一 Python 腳本動態生成單文件 HTML。雙擊運行，瀏覽器秒開，立刻進入工作狀態。這就是極客該有的優雅。

### 2. 🔒 隱私至上，數據絕對安全 (Privacy First)
你的資料，你的電腦。**所有的對話紀錄、工作區狀態以及 API Key，都完全保存在你本地瀏覽器的 `LocalStorage` 中**。不會有任何中間伺服器攔截你的專有程式碼 (Proprietary Code)，完美符合企業與接案開發者的最高合規需求。

### 3. 💻 專為開發者武裝 (Built for Coders)
這不是一個玩具聊天視窗，而是一個硬核的程式碼助手：
* **代碼高亮與一鍵複製**：內建 `highlight.js` 支援，無論多複雜的程式碼區塊都能完美渲染，並附帶一鍵 Copy 功能。
* **Token 本地精準估算**：內建自研的 Token 估算演算法。即時監控每個工作區的消耗，精準控制 API 成本，讓你的每一分錢都花在刀口上。
* **自訂上下文深度**：在設定中自由調整攜帶的歷史對話數量，拒絕無效的 Token 浪費。

### 4. 🎨 沉浸式科技感 UI (Cyber-Tech Design)
採用深色極客主題（Dark Mode）搭配專屬科技綠（Tech-Green）。直覺的側邊欄管理，讓你輕鬆切換不同的 `Agent` 與 `Project` 工作區。提供如同 IDE 般的沉浸式對話體驗。

### 5. 🧠 支援推理模型解析 (Reasoning Support)
原生支援 DeepSeek-Reasoner（Think 模式）的思維鏈（CoT）渲染。AI 的思考過程會被優雅地折疊與區隔展示，讓你能看透 AI 解題的每一步邏輯。

## ⚡ 快速開始 (Quick Start)

只需要**10**秒鐘，啟動你的 AntroCode：

1. **下載腳本**：下載本專案的 `AntroCode_1.py`。
2. **執行腳本**：
   ```bash
   python AntroCode_1.py
開始使用：腳本會自動在當前目錄生成 **antrocode_ui_pro_max.html** 並自動為你打開瀏覽器。

設定 **API Key**：點擊**左下角** ⚙️ Settings，輸入你的 **API Key**，即可開始對話！
