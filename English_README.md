# 🟢 AntroCode - The Ultimate Single-File AI Client

![Version](https://img.shields.io/badge/version-1-00FFAA?style=for-the-badge&logoColor=black)

**AntroCode** is an ultra-lightweight, out-of-the-box, pure frontend LLM client. No bloated `node_modules`, no backend server deployment required. Just run a single Python script to instantly get a cyberpunk-styled AI workspace designed specifically for developers.

Currently natively supports **DeepSeek API (Think / Fast modes)**.

---

## 🔥 Why Choose AntroCode? (Core Features)

We stripped away all complex infrastructure, leaving only pure productivity and absolute data security:

### 1. 🚀 Ultra-Lightweight & Out-of-the-Box (Zero-Install)
Tired of `npm install`ing hundreds of megabytes of dependencies every time? AntroCode dynamically generates a single-file HTML via a single Python script. Double-click to run, instantly opens in your browser, and you're ready to work. This is the elegance geeks deserve.

### 2. 🔒 Privacy First, Absolute Data Security
Your data, your machine. **All chat history, workspace states, and API Keys are stored entirely in your local browser's `LocalStorage`.** No intermediary servers will ever intercept your proprietary code, perfectly meeting the highest compliance requirements for enterprises and freelance developers.

### 3. 💻 Built for Coders
This isn't a toy chat window; it's a hardcore coding assistant:
* **Syntax Highlighting & One-Click Copy**: Built-in `highlight.js` support perfectly renders even the most complex code blocks, complete with a one-click Copy button.
* **Precise Local Token Estimation**: Features a proprietary local token estimation algorithm. Monitor usage in real-time for each workspace to precisely control API costs, ensuring every penny is well spent.
* **Customizable Context Depth**: Freely adjust the number of historical messages included in the context via settings, eliminating wasted tokens.

### 4. 🎨 Immersive Cyber-Tech UI (Cyber-Tech Design)
Features a Dark Mode geek theme paired with a signature Tech-Green. Intuitive sidebar management lets you easily switch between different `Agent` and `Project` workspaces, providing an immersive, IDE-like chat experience.

### 5. 🧠 Reasoning Support
Native support for rendering the Chain of Thought (CoT) of DeepSeek-Reasoner (Think mode). The AI's thought process is elegantly collapsed and separated, allowing you to see through every logical step of the AI's problem-solving.

## ⚡ Quick Start

Launch your AntroCode in just 3 seconds:

1. **Download the script**: Download `AntroCode_02.py` from this repository.
2. **Run the script**:
   ```bash
   python AntroCode_02.py
Start using: The script will automatically generate antrocode_ui_pro_max.html in the current directory and open your browser for you.

Set API Key: Click ⚙️ Settings in the bottom left corner, enter your API Key, and start chatting!
