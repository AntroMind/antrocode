## 🔥 为什么选择 AntroCode？ (Core Features)

我把所有复杂的基础设施全部抛弃，只留下最纯粹的生产力与绝对的资料安全：

### 1. 🚀 极致轻量，开箱即用 (Zero-Install)
厌倦了每次都要 `npm install` 几百 MB 的依赖？ AntroCode 透过单一 Python 脚本动态生成单文件 HTML。双击运行，浏览器秒开，立刻进入工作状态。这就是极客该有的优雅。

### 2. 🔒 隐私至上，数据绝对安全 (Privacy First)
你的资料，你的电脑。 **所有的对话纪录、工作区状态以及 API Key，都完全保存在你本地浏览器的 `LocalStorage` 中**。不会有任何中间伺服器拦截你的专有程式码 (Proprietary Code)，完美符合企业与接案开发者的最高合规需求。

### 3. 💻 专为开发者武装 (Built for Coders)
这不是一个玩具聊天视窗，而是一个硬核的程式码助手：
* **代码高亮与一键复制**：内建 `highlight.js` 支援，无论多复杂的程式码区块都能完美渲染，并附带一键 Copy 功能。
* **Token 本地精准估算**：内建自研的 Token 估算演算法。即时监控每个工作区的消耗，精准控制 API 成本，让你的每一分钱都花在刀口上。
* **自订上下文深度**：在设定中自由调整携带的历史对话数量，拒绝无效的 Token 浪费。

### 4. 🎨 沉浸式科技感 UI (Cyber​​-Tech Design)
采用深色极客主题（Dark Mode）搭配专属科技绿（Tech-Green）。直觉的侧边栏管理，让你轻松切换不同的 `Agent` 与 `Project` 工作区。提供如同 IDE 般的沉浸式对话体验。

### 5. 🧠 支援推理模型解析 (Reasoning Support)
原生支援 DeepSeek-Reasoner（Think 模式）的思维链（CoT）渲染。 AI 的思考过程会被优雅地折叠与区隔展示，让你能看透 AI 解题的每一步逻辑。

## ⚡ 快速开始 (Quick Start)

只需要**10**秒钟，启动你的 AntroCode：

1. **下载脚本**：下载本专案的 `AntroCode_1.py`。
2. **执行脚本**：
```bash
python AntroCode_1.py
```
开始使用：脚本会自动在当前目录生成 **antrocode_ui_pro_max.html** 并自动为你打开浏览器。

设定 **API Key**：点击**左下角** ⚙️ Settings，输入你的 **API Key**，即可开始对话！
